## Notes:
- Commit color density should be relative to the highest day activity and broken into 4 quarters.

- For example if the single day activity is 84 commits the breakdown of color density would be:

Darkest: 63-84
Darker: 42-62
base: 21-41
Lighter: 1-20
lightest: 0
